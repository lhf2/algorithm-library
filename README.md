# 使用JS来实现自己的算法库

## 栈
- [x] 添加元素：push(element)
- [x] 删除元素：pop()
- [x] 查看栈顶元素：peek()
- [x] 判空：isEmpty()
- [x] 元素个数：size()
- [x] 清空：clear()
- [x] 输出字符串：toString()


## 队列

### 普通队列
- [x] 添加元素：enqueue(element)
- [x] 删除元素：dequeue()
- [x] 查看队头元素：peek()
- [x] 判空：isEmpty()
- [x] 元素个数：size()
- [x] 清空：clear()
- [x] 输出字符串：toString()

### 双端队列
- [x] 队列前端添加元素：addFront(element)
- [x] 队列后端添加元素：addBack(element)
- [x] 队列前端删除元素：removeFront()
- [x] 队列后端删除元素：removeBack()
- [x] 返回队列前端元素：peekFront()
- [x] 返回队列后端元素：peekBack()
- [x] 判空：isEmpty()
- [x] 元素个数：size()
- [x] 清空：clear()
- [x] 输出字符串：toString()
